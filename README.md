## 📱 Как настроить прокси для телеграма на андроид (Android) и айфон (iPhone / iOS)

Пошаговые инструкции, как настроить прокси:
1. Откройте приложение Telegram.
2. Зайдите в **Настройки** → **Данные и память**.
   
###  ![1б](https://github.com/user-attachments/assets/ac77de9b-f9a4-4f90-9b48-6030bde68aa9)

----
----
4. Прокрутите вниз и выберите **Прокси** → **Настройки прокси**.

###  ![2б](https://github.com/user-attachments/assets/221a1bf6-c6ee-402c-b060-fe2c7d258373)


----
----   
5. Сдвиньте переключатель **Автопереключение прокси** вправо.
